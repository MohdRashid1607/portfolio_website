# 🚀 Personal Portfolio Website

Welcome to my personal portfolio repository! This is a modern, responsive, and highly interactive developer portfolio built to showcase my projects, academic modules, skills, and professional experience.

## ✨ Key Features
- **Responsive Design:** Fully mobile-friendly layout ensuring a seamless experience across all devices.
- **Dark/Light Mode:** Integrated theme toggle with `localStorage` persistence so user preference is saved.
- **Dynamic Animations:** Scroll-reveal animations, typing text effects, and animated skill tracking bars.
- **Project Case Studies:** 15 individual, highly detailed project pages demonstrating modules ranging from Creative Coding to Machine Learning.
- **Functional Contact Form:** Fully integrated with Web3Forms to send emails directly to my inbox without a backend server.
- **Interactive Carousels:** Utilizes Swiper.js for smooth testimonials and certificates sliders.

## 🛠️ Built With
- **HTML5 & CSS3** (Custom Grid/Flexbox layouts, Vanilla CSS, CSS Variables for theming)
- **Vanilla JavaScript** (DOM manipulation, Fetch API, Intersection Observers)
- **Swiper.js** (For interactive sliders)
- **ScrollReveal.js** (For scroll-triggered animations)
- **Boxicons** (For modern typography icons)
- **Web3Forms API** (For contact form submission)

## 🚀 Deployment (Vercel)
This website is optimized for incredibly fast static hosting. 
1. Connect this repository to your **Vercel** dashboard.
2. Leave the "Build Command" and "Output Directory" blank (as this is pure HTML/CSS/JS).
3. Click **Deploy**.

## 📝 Next Steps Post-Deployment
- Update the **Web3Forms allowed domains** with the live Vercel URL so the contact form accepts submissions in production.
- Upload a `CV.pdf` file to the root directory to activate the "Download CV" button.
