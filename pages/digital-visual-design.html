// Overview

About This Module

Digital Visual Design is the creative symphony of digital elements — a synchronised ballet of colours, typography, imagery, and layout inside digital spaces. It's the intentional creation of visual components to evoke feelings, convey messages, and influence user experiences.

This module introduced the core principles of visual communication — from understanding how users perceive information, to building real website redesigns in Canva and Figma.

"Problems with visual design can turn users off so quickly that they never discover all the smart choices you made with navigation or interaction design."

— Jesse James Garrett

The module was structured as a reflective portfolio — each week building on the last. Starting from atomic design elements (dots, lines, shapes) all the way up to full website redesigns for BathSpa RAK University.

The key insight: "Every object on a digital screen is a design decision." Learning to make those decisions intentionally — not accidentally — was the heart of this module.

Deliverables included weekly reflective commentaries, design exercises, A4 poster creation, reverse wireframing, page tracing, and two website redesigns — all documented in a Microsoft Sway portfolio.

// Core Concepts

Principles Mastered

Colour Theory

The psychological impact of colour in design — how it evokes emotion, guides attention, and communicates brand identity.

Typography

Effective typography improves readability, communicates tone, and preserves visual appeal. Essential for user participation and impactful communication.

Contrast & Hierarchy

The right contrast and hierarchy direct user attention, prioritise content, and create a smooth user experience that feels effortless.

Gestalt Psychology

Users perceive visuals as cohesive wholes. Grouping similar elements through proximity and similarity makes interfaces more readable and intuitive.

Imagery

The right imagery communicates ideas, generates emotion, and creates an eye-catching experience. Selective visuals improve the design's overall impact.

Form Follows Function

Every layout decision is guided by content objectives. Design must be deliberate and purposeful — never decorative for its own sake.

// Reflection

Key Learnings

01

Every design element is a deliberate decision

From button colour to font weight — nothing should be accidental. This module shifted my mindset to always ask "why" before placing anything on screen.

02

Gestalt psychology makes interfaces intuitive

Understanding how users group and perceive information changed how I approach layout. Proximity, similarity, and continuity are invisible forces that make or break a UI.

03

Visual design is the first layer of trust

Bad visual design turns users off before they reach your features. First impressions are visual, not functional — and this portfolio itself is proof of that principle.

04

Reverse wireframing reveals invisible structure

Breaking down existing designs taught me to see beneath the surface of any UI and understand the structural logic behind professional layouts.

05

Canva and Figma serve different stages

Canva is fast and expressive — great for quick visual work. Figma is precise and built for professional product design, prototyping, and collaboration.
