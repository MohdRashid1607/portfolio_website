About This Module

Web Development 1 was the module where the web became real — not as something to consume, but as something to build. Starting from the very first HTML tag and ending with interactive JavaScript applications, this module laid the technical foundation for everything I've built since.

Each of the six chapters introduced a new layer of front-end development — from HTML structure and CSS styling, through multi-page navigation and media embedding, all the way to writing JavaScript logic for a petrol calculator and an RGB colour guessing game.

"Every website you have ever visited was built with the same three languages. Understanding HTML, CSS, and JavaScript isn't just technical skill — it's learning to read and write the language of the web."

— Web Development 1, M.Rashid

// Reflection

Key Learnings

01

HTML, CSS, and JavaScript each do one job — and do it well

The separation of structure (HTML), presentation (CSS), and behaviour (JavaScript) is one of the most elegant design decisions in web technology. Understanding why they're separate makes you better at using all three.

02

The bonus exercises built more confidence than the mandatory ones

The RGB colour guessing game and the multi-page movie site pushed beyond the brief. Choosing to do bonus work meant wrestling with harder problems — and the extra struggle produced extra skill.

03

Layout is the hardest skill to develop

Chapter 4 (Page Layout) was the most challenging — understanding how elements flow, stack, and interact on screen requires a mental model that takes real practice to build. Flexbox clicked slowly, then all at once.

04

JavaScript makes the web feel alive

The shift from static HTML/CSS to interactive JavaScript was the most exciting moment in the module. Writing a function that responds to user input — the petrol calculator, the colour game — made the browser feel like a canvas, not a document.

05

This module is why I build websites today

Web Development 1 is directly responsible for the portfolio you're reading right now. Every HTML element, CSS property, and JavaScript function used on this site traces back to the foundations built in this module.
